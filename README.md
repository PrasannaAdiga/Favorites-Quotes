# Favorite Quotes

## This Application is created by Ionic 3 with Angular 5

### This Application provides the following functionalities

#### It provides a library, where user can find list of existing quotes
#### User can choose any quote and mark it as a Favorite Quote 
#### User can un-favorite any favorite quote.
